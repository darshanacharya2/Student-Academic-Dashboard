# Student Academic Dashboard

A comprehensive, role-based academic portal designed for educational institutions. The application provides dedicated, sleek interfaces for Students, Staff (Teachers), Head of Departments (HOD), and Super Admins.

## 🌟 Key Features

*   **🎓 Student Portal**
    *   View real-time attendance and internal assessment marks.
    *   Simulated assignment submission with drag-and-drop file support.
    *   Track achievements and receive targeted announcements.
*   **👨‍🏫 Staff Portal**
    *   Mark and track student attendance securely.
    *   Create assignments, view submissions, and grade them.
    *   Export attendance and grade reports directly to CSV.
*   **🏢 HOD Portal**
    *   Monitor department-wide analytics and performance charts.
    *   Approve or reject Internal Assessment (IA) marks submitted by staff.
    *   Broadcast announcements to specific batches or the entire department.

## 🛠️ Tech Stack

*   **Frontend Framework:** React 18 with Vite
*   **Language:** TypeScript
*   **Styling:** Tailwind CSS
*   **UI Components:** Radix UI (shadcn/ui inspired) + Lucide Icons
*   **Charts:** Recharts
*   **State Management:** React Context + LocalStorage Persistence

## 🚀 Getting Started

To run this project locally:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/darshanacharya2/Student-Academic-Dashboard.git
    cd Student-Academic-Dashboard
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Start the development server:**
    ```bash
    npm run dev
    ```
4.  Open your browser and navigate to the URL provided by Vite (usually `http://localhost:5173`).
